call it spring blog
client: Aldo Group
designer: Michael R Saravia

The project was to create a completely new website from scratch for the Call It Spring brand. The purpose of the website was to allow authors to post articles, galleries, videos, have Instagram & Facebook feed sections and promote events in an amazing youthful look and experience. I designed the project with a responsive theme and Pinterest look. I created icons for categories, sections and also used social network icons to guide users through the site as breed crumbs tools.

My favourite part of this project was coming up with the concept of having the background fixed in the browser and only moving the content up or down with die-cut boxes where the background would peak through.
